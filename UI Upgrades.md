# kDOUGH UI Upgrades

## Status Tracker

### A. Visual Design System Overhaul
- [x] **A1. Glassmorphism Card Style** — Frosted-glass backgrounds with `backdrop-blur` and subtle transparency on nav, day cards, modals
- [x] **A2. Gradient Accents** — Replace flat amber buttons/accents with subtle warm gradients (amber → orange)
- [x] **A3. Dark Mode** — Full dark mode toggle with smooth transition
- [x] **A4. Refined Typography** — Upgrade font pairing with a display font for headings

### B. Navigation & Layout
- [x] **B5. Bottom Tab Bar (Mobile)** — Persistent bottom tab bar replacing hamburger menu
- [x] **B6. Sidebar Navigation (Desktop)** — Slim sidebar with icons + labels on larger screens
- [x] **B7. Bento Grid Dashboard** — Bento-style layout with calendar + stats blocks

### C. Weekly Calendar Improvements
- [x] **C8. Drag & Drop Meals** — Drag recipes between days and meal slots
- [x] **C9. Meal Slot Expansion on Click** — Inline expand with animation instead of modal
- [x] **C10. Visual Meal Indicators** — Icon badges on filled meal slots (prep time, etc.)
- [x] **C11. Week Summary Bar** — Progress bar showing meals planned count

### D. Micro-Interactions & Animation
- [x] **D12. Toast Notifications** — Animated toast popups for action feedback
- [x] **D13. Card Hover Lift** — Subtle translateY + shadow on hover
- [x] **D14. Smooth Page Transitions** — Route transition animations
- [x] **D15. Skeleton Loading Upgrade** — Shimmer-effect skeletons

### E. Recipe Page Enhancements
- [x] **E16. Recipe Card Images** — Image support on recipe cards and calendar
- [x] **E17. Recipe Card Redesign** — Larger cards with image area and color stripe
- [x] **E18. Quick-Add Recipe FAB** — Floating action button for recipe creation

### F. Grocery List Polish
- [x] **F19. Swipe-to-Check (Mobile)** — Swipe gestures for check/delete
- [x] **F20. Aisle Grouping Icons** — Category icons next to section headers
- [x] **F21. Smart Sort Toggle** — Toggle between category/alphabetical/recipe grouping

### G. Global UX
- [x] **G22. Onboarding Flow** — 3-step guided tour for first-time users
- [x] **G23. Empty State Illustrations** — Small illustrations for empty states
- [x] **G24. Pull-to-Refresh (Mobile/PWA)** — Native pull-to-refresh gesture
- [x] **G25. Keyboard Shortcuts** — Power-user keyboard shortcuts
